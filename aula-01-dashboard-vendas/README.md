# Dashboard de Vendas (Power BI)

## Preview

![Dashboard](preview.png)

---

## Objetivo

Desenvolver um dashboard para análise de desempenho de vendas, permitindo identificar padrões de receita, comportamento ao longo do tempo e diferenças de performance entre regiões e marcas.

---

## Principais Análises

* Faturamento total consolidado
* Evolução da receita ao longo do tempo
* Comparação de desempenho por região
* Análise das principais marcas por faturamento

---

## Visualizações

O dashboard inclui:

* Gráfico temporal para acompanhamento da receita ao longo dos períodos
* Comparativos por região para análise de distribuição geográfica
* Ranking de marcas com base no faturamento
* Filtros interativos para exploração dos dados

---

## Tecnologias Utilizadas

* Power BI
* DAX (Data Analysis Expressions)
* Excel

---

## Insights Possíveis

* Identificação de regiões com maior contribuição no faturamento
* Concentração de receita em marcas específicas
* Variações de desempenho ao longo do tempo
* Possíveis oportunidades de expansão em regiões com menor representatividade
