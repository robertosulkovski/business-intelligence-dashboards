# Dashboard de RH (Power BI)

## Preview

![Dashboard RH](./preview.png)

---

## Objetivo

Desenvolver um dashboard interativo para monitoramento de indicadores de Recursos Humanos, permitindo acompanhar movimentações de colaboradores, analisar a estrutura organizacional e identificar padrões relacionados à rotatividade e distribuição da força de trabalho.

---

## Principais Indicadores (KPIs)

* Funcionários Ativos: Total de colaboradores atualmente ativos
* Contratações: Volume total de admissões realizadas
* Desligados: Total de colaboradores desligados
* Rotatividade (%): Percentual de turnover com base nas movimentações

---

## Visualizações

O dashboard inclui:

* Gráfico de área para análise da evolução de contratações
* Gráfico de barras para distribuição de colaboradores por cidade
* Gráfico de rosca para análise de gênero
* Visual hierárquico para estrutura organizacional por cargo
* Filtro interativo por cargo

---

## Tecnologias Utilizadas

* Power BI
* DAX (Data Analysis Expressions)
* Modelagem de dados
